React 기반 클라이언트와 Node.js 서버 간의 API 통신 시스템 아키텍처 설계 및 구현.

<br>

---

- **Frontend**
    
    **React.js**
    
    - 역할: UI 제공 및 사용자 입력 처리
    
    - 호스팅: Nginx를 사용하여 정적 파일 제공
 
<br>

- **Backend**
    
    **Node.js** 
    
    - 프레임워크: Express를 사용하여 RESTFUL API 구현
    
    - 역할: 클라이언트 요청 처리
